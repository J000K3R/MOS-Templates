# 📦 J000K3R's MOS Templates

Templates for the **MOS Hub** – a collection of templates that are **directly discoverable inside the MOS Hub**.

> 📖 **Full documentation, template guides and setup instructions:**
> **[j000k3r.github.io/MOS-Templates](https://j000k3r.github.io/MOS-Templates/)**

---

## 🚀 Overview

MOS Templates integrate directly with the **MOS Hub** and provide:

- 🐳 Discoverable **Docker templates** inside the MOS Hub
- 🧩 Reusable **plugin templates** for MOS extensions
- 📦 Ready-to-use **docker-compose stacks** for common services
- ✅ A standardized way to publish and maintain templates

---

## 📂 Repository Structure

| Folder / File | Description |
|---|---|
| `docker/` | Docker container templates for MOS Hub |
| `compose/` | docker-compose stacks shown in MOS Hub |
| `plugins/` | MOS plugin templates |
| `icons/` | Icons and graphical assets for Hub display |
| `maintainer.json` | Maintainer & metadata information |
